Core ArchitectureArquitectura Baseコアアーキテクチャ

Built on Void Linux with runit for lightning-fast boot times. Features the MATE desktop environment with essential tools pre-installed.

Basado en Void Linux con runit para tiempos de arranque ultrarrápidos. Incluye el entorno de escritorio MATE con herramientas esenciales preinstaladas.

Void Linuxをベースに、runitで 超高速ブートを実現。MATEデスクトップ環境と 必須ツールを事前インストール済み。

  • Flatpak - Universal packages
  • Tinyfetch - System info
  • btop - Resource monitor
  • Octoxbps - GUI package manager

Optimizado para máximo rendimiento con soporte completo Vulkan y drivers non-free de Intel/AMD preconfigurados para gaming sin problemas desde el primer día.

完全なVulkanサポートと、事前設定された Intel/AMD 用 non-free ドライバーで、初日からシームレスな ゲーミング体験を提供。

  • Pre-configured GPU drivers
  • Optimized performance settings
  • Ready for Steam & Lutris

Why it was createdPor qué se creó作成理由

The project originally started as a personal hobby. Over time, it evolved into a structured project aimed at offering an easy-to-install and user-friendly Void Linux version for the end user.

El proyecto se originó inicialmente como un hobby personal. Con el tiempo, evolucionó hasta convertirse en un proyecto estructurado con el objetivo de ofrecer una versión de Void Linux fácil de instalar y utilizar para el usuario final.

このプロジェクトは元々個人的な趣味として始まりました。 時を経て、エンドユーザー向けに簡単にインストールできて 使いやすい Void Linux バージョンを提供することを目的とした 構造化されたプロジェクトへと進化しました。

Target AudiencePúblico dirigido対象ユーザー

This distribution is aimed at users who want a lightweight and robust operating system, easily customizable for their specific workflow without the usual complications of a manual setup from scratch.

Esta distribución está dirigida a usuarios que desean un sistema operativo ligero y robusto, fácilmente adaptable a su flujo de trabajo específico sin las complicaciones habituales de una configuración manual desde cero.

このディストリビューションは、軽量で堅牢なオペレーティングシステムを求め、 一から手動でセットアップする際の一般的な複雑さなしに、 特定のワークフローに簡単にカスタマイズしたいユーザーを対象としています。
